# Chapter 07 | Loops in Python

Loops are used to execute a block of code repeatedly. Instead of writing the same code multiple times, loops help automate repetitive tasks, making programs shorter, cleaner, and more efficient.

Python provides two main types of loops:

- **while loop**
- **for loop**

---

# While Loop

A **while loop** repeatedly executes a block of code **as long as the given condition remains `True`**.

The condition is checked **before** each iteration. If the condition becomes `False`, the loop stops.

A `while` loop is useful when the number of iterations is **not known beforehand**.

---

## Syntax

```python
while condition:
    # code to execute
```

---

## Example

```python
count = 1

while count <= 5:
    print(count)
    count += 1
```

### Output

```
1
2
3
4
5
```

---

## Infinite Loop

If the loop condition never becomes `False`, the loop runs forever.

### Example

```python
while True:
    print("This will run forever.")
```

> **Note:** Always ensure that the loop condition eventually becomes `False` unless an infinite loop is intentionally required.

---

# For Loop

A **for loop** is used to iterate over a sequence such as a string, list, tuple, set, dictionary, or range.

Unlike the `while` loop, a `for` loop is generally used when the number of iterations is known.

---

## Syntax

```python
for variable in sequence:
    # code to execute
```

---

## Example

```python
fruits = ["Apple", "Banana", "Mango"]

for fruit in fruits:
    print(fruit)
```

### Output

```
Apple
Banana
Mango
```

---

## Example with String

```python
word = "Python"

for letter in word:
    print(letter)
```

### Output

```
P
y
t
h
o
n
```

---

## Range Function in Python

The `range()` function generates a sequence of numbers. It is commonly used with `for` loops.

The sequence generated by `range()` **does not include the ending value**.

---

## Syntax

### One Argument

```python
range(stop)
```

Starts from `0` and ends at `stop - 1`.

Example:

```python
for i in range(5):
    print(i)
```

Output

```
0
1
2
3
4
```

---

### Two Arguments

```python
range(start, stop)
```

Example

```python
for i in range(2, 6):
    print(i)
```

Output

```
2
3
4
5
```

---

### Three Arguments

```python
range(start, stop, step)
```

Example

```python
for i in range(1, 10, 2):
    print(i)
```

Output

```
1
3
5
7
9
```

---

# For Loop with Else

## Theory

Python allows an `else` block to be used with a `for` loop.

The `else` block executes **after the loop finishes normally**.

If the loop is terminated using a `break` statement, the `else` block is **not executed**.

---

## Syntax

```python
for item in sequence:
    # code
else:
    # executes after loop finishes
```

---

## Example

```python
for i in range(5):
    print(i)
else:
    print("Loop completed successfully.")
```

### Output

```
0
1
2
3
4
Loop completed successfully.
```

---

# Break Statement

## Theory

The `break` statement immediately terminates the loop.

Once `break` is executed, Python exits the loop and continues with the next statement after the loop.

---

## Example

```python
for i in range(1, 6):
    if i == 4:
        break
    print(i)
```

### Output

```
1
2
3
```

---

# Continue Statement

## Theory

The `continue` statement skips the current iteration and moves directly to the next iteration of the loop.

It does **not** terminate the loop.

---

## Example

```python
for i in range(1, 6):
    if i == 3:
        continue
    print(i)
```

### Output

```
1
2
4
5
```

---

# Pass Statement

## Theory

The `pass` statement is a placeholder that does nothing.

It is used when a statement is syntactically required, but you do not want to write any code yet.

---

## Example

```python
for i in range(5):
    if i == 3:
        pass
    print(i)
```

### Output

```
0
1
2
3
4
```

---

## Another Example

```python
if True:
    pass
```

This program runs without any error, even though no action is performed.

---

# Difference Between `break`, `continue`, and `pass`

| Statement | Purpose | Effect |
|-----------|---------|--------|
| `break` | Terminates the loop immediately | Exits the loop |
| `continue` | Skips the current iteration | Moves to the next iteration |
| `pass` | Placeholder statement | Does nothing |

---

## `end` Parameter in Python

The `end` parameter in the `print()` function specifies what should be printed after the output. By default, `print()` ends with a newline (`\n`), which moves the cursor to the next line.

### Syntax

```python
print(value, end="")
```

### Example

```python
print("Hello", end="")
print("World")
```

**Output:**

```text
HelloWorld
```

> **Note:** `end=""` removes the default newline, allowing the next `print()` statement to continue on the same line.

---

# Summary

Loops are essential for automating repetitive tasks in Python. The `while` loop is ideal when the number of iterations is unknown, while the `for` loop is preferred for iterating over sequences or a known range of values. Python also provides control statements such as `break`, `continue`, and `pass` to modify loop behavior, making programs more flexible, readable, and efficient.
