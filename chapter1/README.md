# Chapter 1 | Modules, Comments $ pip

A module is a file containing code written by someone else(usually) which can be imported and used in our program
Pip is a package manager for pyhton. We use it to install a module on our system

>>Two types of modules

1. Build in Modules (preinstalled in python) | EG: OS, random etc.
2. External Modules (Needed to install using pip) | eg: tensorflow, flask etc.

>>Using Python as calculator

```We can use python as calculator by typing "python"+Enter in terminal. This opens REPL (Read Evaluate Print Loop).```

>>Comments 

comments are used to write something that the programmer do not want to execute.

>>Two types of comments

1. Single line comment wirtten using "#" at the start of the line.
2. Multi-Line comment written using "#" at each line or using ("""......""").
