 # Chapter 2 | Variables and Data Types

A variable is a name given to a memory location in a program.

>>Variables

Container to store value.

>>keywords

reserved words in python.

>>identifires

class/function/variable_name.

>>Data Types | few data types:
1. Integers
2. Floating point numbers
3. Strings
4. Booleans
5. None
Python automatically identifies the type of data when we initialize a value.

>>Rules for defining a variable name

- A variable name can contain alphabets, digits, and underscores.
- A variable name can only start with alphabet and underscores.
- No white space is allowed to be used inside a variable name.

>>Operators in Python | Few operators are:

1. Arithmetic operators: +, -, /, *, **, // etc.
2. Assignment operators: =, +=, -= etc.
3. Comparision operators: ==, >, <, >=, != etc.
4. Logical operators: and, or, not.

>> Type() Function and TypeCasting

type() function is used to find the data type of a given variable in python.

>>Functions to convert one data type into another:

A number can be converted into string and vice versa.
1. str()
2. int()
3. float()
and so on..

>> Input() Function

This function allows user to take input from the keyboard as a string.
